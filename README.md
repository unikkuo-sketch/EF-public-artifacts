# EF public artifacts

Public mirror of the latest emission factor check outputs from [emissionfactor](https://github.com/unikkuo-sketch/emissionfactor).

Updated automatically by GitHub Actions after each biweekly check:

- `latest/EF_output.csv`
- `latest/report.json`
- `latest/metadata.json`
- `latest/attachments/` (source files when updates are detected)

Power Automate downloads from raw URLs under `latest/` for OneDrive sync and Teams notifications.
